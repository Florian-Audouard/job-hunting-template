# Standards CV & lettre de motivation en France (2025–2026)

Synthèse de recherche en ligne (juin 2026) servant de référence aux thèmes de ce
dépôt. Sources en bas de page.

## TL;DR

- **Lettre de motivation** : format **sobre et standardisé**, pas de « thème »
  graphique. Une page, structure classique, corps en **Vous / Moi / Nous**.
- **CV** : un thème **soigné** aide, mais le look « moderne » **deux colonnes**
  est risqué pour les **ATS**. Le meilleur compromis = **une seule colonne**,
  hiérarchie typographique forte, **couleur d'accent sobre** → lisible pour
  l'humain *et* parsable par les ATS.
- **Format d'envoi** : **PDF** (texte sélectionnable). Pour les grandes
  entreprises, prévoir éventuellement une variante **DOCX** simple (très ATS-safe).

## Le CV

### Mise en page
- Design épuré, sections claires, **idéalement 1 page**.
- **Expérience avant formation** (les compétences acquises en poste priment).
- Titre de CV accrocheur + courte accroche/profil en haut.
- Couleurs **sobres** : bleu foncé, gris foncé, vert sapin, noir.
- Polices lisibles : **Calibri, Arial, Helvetica** (Liberation Sans = équivalent libre).
- Mettre en avant les **compétences numériques / outils**.

### Ordre de sections recommandé (single-column, ATS)
1. Coordonnées (en-tête)
2. Titre + accroche courte
3. Compétences
4. Expériences professionnelles
5. Formation
6. Langues / certifications / projets

## ATS (Applicant Tracking Systems)

### Pénétration en France (APEC 2025)
- **~27 %** de l'ensemble des entreprises utilisent un ATS.
- **68–75 %** dans les **grandes entreprises** (>200 salariés).
→ Si vous visez de grands groupes, optimiser ATS est important ; pour des PME /
candidatures directes, le rendu visuel prime davantage.

### Pourquoi les deux colonnes posent problème
L'ATS lit souvent **de gauche à droite, de haut en bas**. Sur deux colonnes, une
compétence de gauche peut se coller à une date de droite, ou une expérience être
séparée de ses missions → texte mélangé, mal indexé.

### Checklist CV ATS-safe
- [ ] **Une seule colonne**.
- [ ] Pas de tableaux, zones de texte, colonnes flottantes.
- [ ] Pas d'info essentielle dans une **image, un pictogramme ou une jauge**.
- [ ] Titres de sections **standards** et explicites.
- [ ] Police classique, **texte PDF sélectionnable** (pas un scan/image).
- [ ] Pas de photo pour les infos critiques (coordonnées en texte).
- [ ] PDF texte **ou** DOCX simple.

## La lettre de motivation

### Mise en page
- **1 page recto**, police lisible (Arial/Calibri) **12–14 pt**.
- Marges ~2,5 cm haut/bas, ~3 cm gauche/droite.
- **PDF** pour figer la mise en page.

### Structure
1. **Expéditeur** en haut à gauche (prénom, nom, adresse, téléphone, email).
2. **Destinataire** en haut à droite (entreprise, « À l'attention de … », adresse).
3. **Date** du jour.
4. **Objet** clair (poste visé + référence de l'offre).
5. **Corps** — méthode **Vous / Moi / Nous** :
   - *Vous* : pourquoi cette entreprise vous intéresse (1–2 éléments précis).
   - *Moi* : vos compétences/expériences en lien avec le poste.
   - *Nous* : projection de la collaboration future.
6. **Formule de politesse** + signature.
- Longueur : **15–25 lignes / 150–300 mots**.

## Spec de design appliquée dans ce dépôt
Thème CV **single-column ATS-safe** commun aux 5 formats :
- 1 colonne, pas de tableaux/icônes/jauges pour l'essentiel.
- Accent sobre **bleu foncé `#1A4F8A`** (titres de section + filet).
- En-tête : Nom (grand) · Titre · ligne de contact en texte simple.
- Ordre : En-tête → Profil → Compétences → Expériences → Formation → Langues.
- Dates alignées à droite, ~1 page, texte sélectionnable.
